# 🐱 NyaClicker

Продвинутый автокликер для Windows 10 с красивым тёмным интерфейсом, нашим неко-маскотом и широкими возможностями настройки.

![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🖱 **5 типов действий** | Клик, Двойной клик, ПКМ, Удержание, Свайп |
| 📍 **Захват координат** | Нажми кнопку «Захват» — кликни по нужному месту |
| ⚡ **Список действий** | Добавляй, редактируй, переупорядочивай, включай/выключай |
| ⏱ **Гибкий тайминг** | Задержка перед действием, длительность удержания, повторения |
| 🔀 **Случайный разброс** | Добавляй рандомность к задержкам для обхода ботозащиты |
| 💾 **Сохранение/загрузка** | Сохраняй и загружай списки действий в JSON |
| 🎮 **Горячие клавиши** | Настраиваемая клавиша Старт/Стоп (по умолчанию F8) |
| 🌙 **Тёмная тема** | Современный интерфейс в тёмных тонах с розовыми акцентами |

## 🚀 Установка

### 1. Клонируй репозиторий

```bash
git clone https://github.com/xristos-dev/NyaClicker.git
cd NyaClicker
```

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. Запусти

```bash
python main.py
```

> **Требования:** Python 3.9+, Windows 10

## 🎮 Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **F8** | Старт / Стоп |
| **📍 Захват** | Кликни по нужному месту на экране (в диалоге) |
| **Мышь в угол (0, 0)** | Аварийная остановка (FailSafe) |

---

## 📖 Использование

1. Нажми **+ Добавить** для создания нового действия
2. Выбери тип: Клик / Двойной клик / ПКМ / Удержание / Свайп
3. Укажи координаты вручную или нажми **📍 Захват** и кликни по нужному месту
4. Настрой задержку, длительность и количество повторений
5. Нажми **▶ Старт** или **F8** для запуска

### Типы действий

- **Клик** — одиночный клик по координатам
- **Двойной клик** — двойной клик по координатам
- **ПКМ** — клик правой кнопкой мыши
- **Удержание** — зажать кнопку мыши на заданное время
- **Свайп** — перетащить от точки A до точки B

### Настройки

| Параметр | Описание |
|----------|----------|
| Повторений цикла | Сколько раз прогнать весь список (0 = бесконечно) |
| Доп. задержка | Глобальная пауза между каждым действием (мс) |
| Случайный разброс | Рандомный ±разброс к задержкам |
| Скорость движения | Длительность движения курсора (0 = мгновенно) |

---

## 💾 Сохранение действий

Списки действий сохраняются в формате `.json` — их можно передавать, делать бэкапы и редактировать вручную.

---

## 🔒 Безопасность

Для экстренной остановки переведи курсор в **верхний левый угол экрана (0, 0)** — сработает FailSafe и кликер остановится немедленно.
